***Delphine Bottarlini -** `delphine.bottarlini@gmail.com`*


**Prérequis**  : L’application Gmail installée sur son téléphone, avec le compte Gmail connecté


**Note** : pour des raisons de sécurité et de vie privée, les captures d’écran ont été anonymisées, ce qui peut conduire à des différences entre ce que vous voyez dans cette procédure et sur votre écran, et aussi mener à quelques incohérences dans le déroulement des captures d’écran

# Principe

- Ajouter une authentification à deux facteurs à son compte Google (avec l’application Google)
- Ainsi, en plus du mot de passe sur le compte Google ou sur la boîte Gmail, une validation sur l’application Google vous sera demandé
- La demande  se produira sur les nouveaux appareils, quand vous changez de pays, ou que vous avez besoin de vous connecter sur un nouvel appareil
- Il est aussi possible de demander le 2FA par SMS

# Etapes

- Une fois connecté sur Gmail sur votre ordinateur, cliquez en haut à droit sur votre avatar, puis cliquez sur “Gérez votre compte Google”. 

![[Pasted image 20260221144930.png]]


Une nouvelle page s’affiche.
- Cliquez à gauche sur “Sécurité”, puis sur “Validation en deux étapes”

![[Pasted image 20260221145120.png]]

- Saisissez votre mot de passe une nouvelle fois

- Cliquez sur “Commencer” dans la fenêtre qui s'affiche : 

![[Pasted image 20260221145207.png]]

- Google va détecter votre téléphone sur lequel vous avez installé l’application Gmail. Si c’est bien le bon téléphone qui s'affichr, cliquez sur “Continuer”, sinon, cliquez sur “Afficher plus d’options” : 
![[Pasted image 20260221145303.png]]

- Une option de secours par SMS ou Appel Téléphonique est demandé. Renseignez votre numéro de téléphone, et choisissez SMS ou Appel téléphonique puis cliquez sur “Envoyez”. Vous allez recevoir un SMS avec un code, renseignez le code dans la nouvelle fenêtre

![[Pasted image 20260221145337.png]]
![[Pasted image 20260221145343.png]]

- Finalement, cliquez sur “Activer”
![[Pasted image 20260221145611.png]]
- La double authentification est maintenant activée sur votre compte, et Google affichera une notification sur l'application Gmail sur le téléphone à chaque fois que vous vous connecterez à Gmail de nouveau.
